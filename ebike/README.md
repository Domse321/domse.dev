# MTB-Routen Hameln

Private E-MTB-Routendatenbank für Domse. Startpunkt: Reimerdeskamp 62, 31787 Hameln. Bike: Bergamont Revox Sport 10.

## Dateien

- `routes.json` – strukturierte Datenbank inkl. Bewertungsmodell
- `index.html` – sortierbares Dashboard
- `gpx/*.gpx` – einfache GPX-Routenpunkte zum Import/Weiterplanen

## Bewertungsmodell

Score = 25% Natur/Schönheit + 18% Trail/Waldweg + 17% wenig Verkehr + 16% E-MTB-Faktor + 12% Aussicht/Highlights + 7% Rundkursqualität + 5% Datenvertrauen.

Wichtig: Die ersten Routen sind kuratierte Starter-Entwürfe. Vor dem Fahren in Komoot/BRouter feinrouten und Trails/Naturschutz beachten.
